# ᚁᛁᚔᛒ ᚅᚎᚔᚏᛁᛋᚔᛒᚈᚁ  

### ᚋᚅᚓ ᚁᚘᚔᚒᚅ ᚁᛅᚈᛂᚉᚅᚁ ᛂᚅ ᚅᚏᛅᚓᛒᛖᚒᛆᚅᛉᚔᚈᚓ

ᚁᛁᚔᛒ ᚅᚎᚔᚏᛁᛋᚔᛒᚈᚁ ᚔᛋ ᚁᛂ ᚅᚐᚉᛖᚅᛂᛉᚔᛒᛅᛏᛂᚅ ᚏᛂᛋᚂᚅᚅᚓᚇ ᚁᛂᚈᚅᚁ ᚅᚒᛁᚔᛋᚄᚄᛔᚋᚔᛋᚋ—
ᚅᚒᛁᚔᛋᚈᚓᚈᚔᚉ–ᚁᚅᛂᛃᚒᚉᚁᚂᛇ ᚅᚒᚏᚂᚅᛖᚅᚅᚁ ᚅᚐᚉᚋᛁᚂᚅᛁᚃᚓ,
ᚅᛖᛏᛋᛁᛚᛁᛒᛋᚔᚍᚉᚅᚔᚅᛉᛂ,
ᛂᚅᛉ  
ᚄᛂᚅᚈᚐᛁᚔᚉ–ᚋᚒᚅ ᛁᚅᛉᛂᚅᚅᚁ.

ᚁᛂᛋᛔᚒᛋᛔᚁᛂᚏᛉ ᛂᚄᚂᛁᚂᛂᛉ ᚁᚎᚓ ᚁᛒᚅᚔᚈᛂ ᚅᛂᛉ ᚋᚅᚓᛋ,
ᚁᚏᚂᛖᚉᛋ,
ᛁᛏᚂᛂᚅᚅᚁ,
ᚏᚒᚏᚁᚈᚐᛂᚅᛋ,
ᚅᛖᛋᚏᛁᚃᛂᛋ
ᚅᚐ ᚁᛁ᚜ᛁ ᚅᚒᛁᚔᛋᚈᚓᚉᚋ.

### **ᚁᛁᚔᛒ–ᚅᚐᛅᚅ**
ᚐᚅ ᚐᚒᛉᛉᛖᛉᛉᛁᚉ ᛁᚅᛇᛋᛃ–ᚂᚐᛉᛁᛉᛉᚅᛉ.

- ᚐᚈᛟ-ᚉᚂᚐᛉᛉᛁᚅᚐᛉᛉᛁᚅ  
- ᚐᚈᛟ-ᚇᛂᚂᚂᛁᚅᚐ
